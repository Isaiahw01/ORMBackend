# E-Commerce Back End
Module 13 Object-Relational Mapping (ORM): E-Commerce Back End

## Installation
```
npm i
```

## Usage
```
mysql -u root -p 
```
when prompted type in your MySQL password
```
source db/schema.sql
exit
npm run seed
node server.js
```

Testing of the code was completed in Insomnia

## Features
* Express.js
* MySQL
* Sequelize
